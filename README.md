## 📖 نظرة عامة

هذا المشروع يشرح بطريقة بصرية ومبسطة **6 مفاهيم أساسية** في عالم نماذج اللغة الكبيرة:

| # | الدليل | الوصف |
|---|--------|-------|
| 01 | **ما هو الـ Token؟** | الوحدة الأساسية التي يفهمها النموذج |
| 02 | **التكلفة والفواتير** | كيف يحتسب عليك كل طلب |
| 03 | **Embeddings و RoPE** | تمثيل النصوص رياضيًا والموضع الدوراني |
| 04 | **قواعد البيانات المتجهية** | البحث الدلالي و RAG |
| 05 | **Attention و QKV** | آلية الانتباه و KV Cache |
| 06 | **عملية الاستدلال** | من الـ Tokens للإجابة النهائية |
